# CEO ORDERS

## Deploy Dashboard

Create `dashboard/index.html` with a mobile-first dark theme dashboard that:
- Shows company health, run count, file count, quality score
- Lists recent actions with status icons
- Displays latest run report
- Shows system logs
- Auto-refreshes every 30 seconds
- Reads data from ceo/memory.json and ceo/reports/ via GitHub raw API

Create `.github/workflows/pages.yml` to deploy dashboard folder to GitHub Pages.

Update README with dashboard URL.

Use this exact HTML structure as base:
- Dark background (#0a0a0f)
- Cyan accents (#00d4ff)
- Cards with rounded corners
- Sticky header
- Fixed refresh button
- No external dependencies
